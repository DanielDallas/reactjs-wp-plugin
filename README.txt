A WordPress plugin to manage and display React components dynamically in WordPress.

== Description ==

The ReactJS Plugin allows users to add, manage, and display React components through a custom post type. You can create components, store their code in a text area, and generate shortcodes to use within any post or page. The plugin provides an intuitive interface for copying and using the shortcodes to embed React components.

== Installation ==

1. Upload the plugin folder to your /wp-content/plugins/ folder.
2. Go to the **Plugins** page and activate the plugin.

== Frequently Asked Questions ==

= How do I add a new React component? =
1. Navigate to **ReactJS Components** in the admin menu.
2. Click **Add New Component**.
3. Enter a title for your component and paste the React code in the provided textarea.
4. Save your component.

= How do I display a React component in a post/page? =
Simply copy the generated shortcode (`[react_component id="X"]`) from the **ReactJS Components** menu and paste it into any post or page. The React component will render where the shortcode is placed.

= How to uninstall the plugin? =
Simply deactivate and delete the plugin from the **Plugins** page.
